📦 Bem-vindo (a)

Um bom controle de estoque pode ser o fator determinante entre fechar uma venda e perdê-la para um concorrente. Afinal, quem não sabe que produtos tem e onde encontrá-los, sofrerá para vendê-los. Por isso, criamos o MQtech Estoque. ❤️

💻 Tecnologias utilizadas:
Angular 8, NodeJS, MongoDB
Deploy: API e banco estão hospedados na Umbler,
Front está hospedado no Github Pages.
